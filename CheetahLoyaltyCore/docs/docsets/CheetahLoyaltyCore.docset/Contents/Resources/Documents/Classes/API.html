  • deleteCredentialsIfNeeded()

    Resets the credentials in the keychain when the app has not yet launched at least once.

    This method also checks for the defaults value of .hasLaunchedOnce. If the defaults value of .hasLaunchedOnce is nil or false, the credentials stored in the keychain will be deleted and a value of true will be stored on the defaults key .hasLaunchedOnce. If the defaults value of .hasLaunchedOnce is true, this just proceeds to read and return the current access token from the keychain.
