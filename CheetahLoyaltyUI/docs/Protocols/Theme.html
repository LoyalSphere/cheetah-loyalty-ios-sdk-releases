Theme

public protocol Theme

Create a new theme by creating a struct that conforms to this protocol. By default accent and text color is automatically computed based on the primary color. If a color of a view does not goes well, just override the paint function that correspond to its view (Use the apperance method for thet view or just leave the overriden method blank)
