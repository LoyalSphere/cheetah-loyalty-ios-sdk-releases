  • paintUIButton(using:)

    Change the color appearance of UIButton.

    The button color style mainly affects buttons inside collection view cells, inside DefaultReloadViewController‘s stack view and buttons inside a CheetahAppWindow. Buttons inside a navigation bar and AlertViewController are set to clear backgrounds with title and tint color set to buttonTintColor.

    The button color style consists of:

    • Outline - This sets the background color of the buttons to clear
    • Solid - This sets the background color of the buttons to primaryColor Both sets the title and tint color to buttonTintColor.

    The default button color style is set to .outline.

    Declaration

    Swift

    open func paintUIButton(using theme: ColorTheme)
