  • handleEmsOpen(_:)

    Handles the EMS Open String.

    The defailt implementation of this method first tries to creates a URL from the emsOpen String, if it the URL initialization returns nil, this method returns. If a URL object is successfully initialized, it is then used to create a URLRequest object with its httpMethod set to "POST". The request is then used to create a URLSessionDataTask using the the shared URLSession. The completion handler of this task calls handleOpen(emsOpenURL:, data:, response:, error: ) if the emsOpen String contains "open", otherwise it calls handleClick(emsOpenURL:, data:, response:, error: ) if theemsOpenStringcontains“click”`. The task is then resumed to be executed.
