  • setupBarcodeIfNeeded()

    Setup barcode if offer is of type barcode.

    This method shows or hides the barcode stack view containing the barcode image view and label. The barcode views will be hidden if the offer type is not a barcode. This also sets the text of the barcode label to the offer’s barcode. Additional setup for the barcode image and other modifications is done by the offer detail delegate.

    Declaration

    Swift

    open func setupBarcodeIfNeeded()
  • setupActionButtons()

    Setup action buttons for the offer detail.

    The action buttons for an offer detail are share and read more for url offer types. The share button is shown when the target url is not blank. The read more button is shown when the offer is of type url and its url is not blank.
