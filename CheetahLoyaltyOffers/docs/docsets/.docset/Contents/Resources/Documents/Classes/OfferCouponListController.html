OfferCouponListController

open class OfferCouponListController : OfferListController

An OfferListController subclass that loads offers of type coupon.

This class also enables clipping of offers of type coupon by default.

The loadParametersdefault value:

["layout": "mobile_list",
"include_clipped": true,
"include_responses": true,
"show_clipped": true,
"result_with_categories": true,
"offer_with_responses": true,
"display_processing_statuses": "pending,clipped",
"has_processing_status": "pending,clipped",
"has_usable": true]
