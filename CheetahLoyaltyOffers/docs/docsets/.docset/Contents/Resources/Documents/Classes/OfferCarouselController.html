OfferCarouselController

open class OfferCarouselController : CarouselCollectionController<Offer>

A subclass of CarouselCollectionController that manages Offer objects in a carousel behavior for a CollectionViewController.

This CarouselCollectionController auto scrolls its content by calling autoScroll(). The auto scroll can be stopped using stopAutoScroll.
