# Cheetah Loyalty iOS SDK

The **Cheetah Loyalty iOS SDK** contains components and tools to build iOS applications using the **Cheetah Loyalty API**.

## Requirements
* Deployment Target: iOS 11.0
* Xcode 11.x
* Swift 5.1
